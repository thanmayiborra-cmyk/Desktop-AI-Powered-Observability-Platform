# AI-Powered Observability Platform

## Overview

AI-Powered Observability Platform is a Python-based monitoring solution that analyzes system metrics, detects incidents, identifies root causes, calculates system health scores, and provides automated recommendations.

## Features

* Incident Detection
* Severity Classification
* Root Cause Analysis
* System Health Score Calculation
* Automated Recommendations
* CSV Data Processing
* Monitoring Summary Report

## Technologies Used

* Python
* CSV
* VS Code
